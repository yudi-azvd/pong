## Dependências
- g++
- CMake
- Make/Ninja 
- SFML


## Baixar, compilar e jogar
    git clone https://github.com/yudi-azvd/pong.git
    cd pong/build
    cmake ..
    make
    ./pong


## Instruções
<kbd>Esc</kbd>: fechar o jogo

### Player 1 (esquerda)
<kbd> W </kbd>: subir

<kbd> S </kbd>: descer

### Player 2 (direita)
<kbd> ↑ </kbd>: subir

<kbd> ↓ </kbd>: descer
